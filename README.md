# Prodigy InfoTech Generative AI Internship - Task 02

## 🖼️ Image Generation with Stable Diffusion

### 📌 Objective

Generate AI images from text prompts using a pre-trained **Stable Diffusion** model. This project demonstrates the capabilities of modern text-to-image generative AI models.

---

## 🛠️ Tools & Technologies

* Python
* Google Colab
* PyTorch
* Diffusers
* Transformers
* Accelerate

---

## ✨ Features

* Uses a pre-trained Stable Diffusion model.
* Converts text prompts into high-quality AI-generated images.
* Supports multiple custom prompts.
* Demonstrates text-to-image generation using Generative AI.

---

## 📝 Sample Prompts

* A futuristic city at sunset with flying cars, ultra realistic, 4k
* A majestic lion sitting on a mountain during sunrise, ultra realistic
* A cyberpunk street at night with neon lights and rain
* A beautiful village surrounded by snowy mountains, photorealistic

---

## 🚀 How to Run

1. Open the notebook in Google Colab.
2. Install the required libraries:

   ```
   pip install diffusers transformers accelerate torch
   ```
3. Run all notebook cells.
4. Enter your desired text prompt.
5. The model will generate an AI image based on the prompt.
