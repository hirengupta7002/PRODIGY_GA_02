# PRODIGY_GA_02

## Task 02 - Image Generation with Pre-trained Models

### Objective
Generate AI images from text prompts using a pre-trained Stable Diffusion model.

### Tools & Technologies
- Python
- Google Colab
- PyTorch
- Diffusers
- Transformers
- Accelerate

### Features
- Uses a pre-trained Stable Diffusion model.
- Generates high-quality images from text prompts.
- Demonstrates text-to-image generation using AI.

### Sample Prompts
- A futuristic city at sunset with flying cars, ultra realistic, 4k
- A majestic lion sitting on a mountain during sunrise, ultra realistic
- A cyberpunk street at night with neon lights and rain
- A beautiful village surrounded by snowy mountains, photorealistic

### Outcome
Successfully generated multiple AI images from different text prompts using Stable Diffusion.
